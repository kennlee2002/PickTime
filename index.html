⏱️ 關於 Picktime 的誕生

一切始於一個課堂裡的天馬行空——「如果我們來做一個購物網站，會是什麼樣子？」

2025 年的深秋，黎明技術學院的一個小團隊開始動了起來。沒有龐大預算，沒有百人工程師，只有三個對設計與技術充滿熱情的人，和一份想把好東西帶給更多人的心意。

Picktime，拾光機——每一件商品，都是值得被時光記住的選擇。我們不追求數量，只追求每一次推薦都讓人心跳一下。

✨ 品牌哲學：光的選品學

你有沒有過那種感覺——在一家店裡，突然某件東西讓你的眼睛亮了？

那一刻，叫做「拾光」。

我們深信，每個人的生活裡都需要幾件這樣的物品：不只是功能，而是溫度。不只是設計，而是故事。Picktime 扮演的角色，是那個走遍千百貨架、幫你挑出那道光的朋友。

從 3C 科技到手工皮件，從北歐燈具到手沖咖啡，每一件登上 Picktime 的商品，都經過我們自己「用過、愛過、捨不得分享」的考驗。這不是演算法推薦，是真正的人工篩選。

🚀 我們的小小野心

我們知道自己只是起步——一個學生專案、一個尚未上線的夢想原型。但我們相信，每個偉大的平台，都曾是某個人的期末作業。

未來的 Picktime，希望成為台灣最懂生活美感的選品平台：讓獨立設計師被看見、讓好東西不再被埋沒、讓每個人都能輕鬆擁有「值得擁有的那道光」。

謝謝你來到這裡，成為我們第一批見證者。✨

📑 畢業專題說明

答辯時可以直接依照此頁介紹系統

一、專題名稱

Picktime Ultimate：AI 智慧電商與會員經營平台。系統以購物流程為核心，加入會員、推薦、後台與客服展示。

  • 品牌名稱：Picktime 拾光機
  • 系統類型：前端電商平台
  • 展示用途：畢業專題與作品集

二、開發動機

現代電商不只是商品展示，而是需要會員經營、推薦系統、訂單追蹤、客服與後台管理，因此本專題以完整平台作為展示目標。

  • 提升購物體驗
  • 模擬真實電商流程
  • 展示前端整合能力

三、使用技術

本版採用 HTML、CSS、JavaScript 製作單頁式展示網站，搭配 localStorage 模擬資料儲存，並保留 Supabase 串接入口。

  • HTML 結構與語意化區塊
  • CSS 玻璃擬態與響應式設計
  • JavaScript 互動與狀態管理

四、核心功能

系統包含首頁、商品列表、商品詳細頁、購物車、會員中心、收藏、優惠券、積分、FAQ、Contact 與展示後台。

  • 商品搜尋與分類
  • 購物車與結帳流程
  • 會員資料與等級制度

五、加分功能

為了讓專題更完整，本版加入 AI 推薦展示、智慧客服、訂單追蹤、商品比較與專題說明頁。

  • 規則式推薦分數
  • 客服自動回覆
  • 物流狀態時間軸

六、未來擴充

未來可以串接真實資料庫、付款金流、物流 API、Email 通知與真正的 AI 模型，讓展示版升級成可部署平台。

  • Supabase 商品與訂單資料表
  • 第三方金流服務
  • AI API 商品推薦

服務說明

購物流程
1
點擊商品卡片

點選任一商品卡片，進入商品詳細頁查看完整介紹、規格與可選樣式。

2
選擇規格 / 樣式

在商品詳情頁選擇你想要的顏色、尺寸、型號或容量，並設定購買數量。

3
加入購物車或直接購買

按下「加入購物車」繼續逛，或「直接購買」快速前往結帳。

4
輸入收件資料並結帳

填寫收件人姓名、手機、地址，選擇付款及配送方式後確認結帳。

⚠️ 此網站為展示版本，結帳後訂單資料儲存於 Supabase，不會產生真實交易或出貨行為。
